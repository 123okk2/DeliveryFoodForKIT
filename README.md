# DeliveryFoodForKIT
금오공대 배달평점
(2019년 1학기 웹 프로그래밍 팀 과제)

## 주제
금오공대 학생들을 위한 배달 후기 및 평점 작성 사이트 제작

## 핵심 기능
1. 가게별 배달 평점 및 후기 기록
2. 후기에 대한 댓글
